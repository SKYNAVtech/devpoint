SKYNAV

DevPoint  ·  Deviation Point Calculator

A professional desktop tool for flight operations engineers. It computes departure deviation points along SID procedures using NavDB or ARINC 424 data. It also has the ability to convert the gross flight path into a net flight path for visualization in Google Earth.

It helps airlines complying with EASA CAT.POL.A.210 Take-off obstacle clearance.

Before first launch

GrtoH Database Setup

SKYNAV DevPoint cannot calculate deviation points without a populated GrtoH table. This is a one-time setup that takes under two minutes.

GrtoH files are not bundled with the installer. They contain simulator specific altitude performance data that you export from your own performance tool.

1
Navigate to %PROGRAMFILES%\SKYNAV DevPoint\Data\GrtoH\ and check Readme.txt.
3
Copy your CSV files into the folder. One file per aircraft . The filename shall be the aircraft database name.
4
Launch SKYNAV DevPoint. The app scans the folder on startup.
Folder structure after setup 
GrtoH\
            ├── AB100A01.csv
            ├── AB200A01.csv
            ├── AB300A01.csv
            └── AB400A01.csv

User Guide

Two modes. One workflow.

Choose the operating mode that fits your available data. Both produce the same CSV and KML output.

RECOMMENDED

NavDB Mode

Uses a full aircraft navigation database export to resolve the complete SID leg sequence including leg types (CF, TF, DF, VA, RF…) and transition waypoints. Provides the highest geometric accuracy for production and regulatory use.

FALLBACK

ARINC424 Mode

When no NavDB is available, derives a straight-out departure track from AptDB runway coordinates and magnetic variation. You can then use your ARINC424 database to complete the SID leg sequence.

Just leave the NavDB CSV empty and the app will automatically switch to ARINC424 mode.

NavDB Mode ARINC424 Mode
Turn geometry Full SID resolution Not modelled
SID routing Yes Straight-out only
Flight phases EO defined by the user EO defined by the user
Splay Area computation Yes Yes
Aircraft selection Yes Yes
